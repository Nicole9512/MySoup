MySQL拾遗
==========================

### 一、使用using
* join语句中，可以使用using来替代on子句的内容。
```
mysql> select * from t1 join t2 using(ID);
```
### 二、MySQL查询语句执行过程
* 客户端 => 连接器 => (缓存 ? 查询缓存 : 分析器) => 优化器 => 执行器。（MySQL8.0之后删掉了查询缓存模块，因为缓存命中率太低，只要表发生更改，整个表的缓存全部失效）
* 以上都执行结束之后，才选择存储引擎，提供读写接口。

### 三、分析器
* 分析器不止检查SQL语法，还检查表、字段是否存在等。
